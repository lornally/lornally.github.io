### 不是那么常用的命令

```sh
gra #缩写, 下面是本尊
git remote add
gpu #缩写, 下面是本尊 #改变git的default remote.
gp -u <remote_name> <local_branch_name>
git init #初始化一个git目录.
#远程覆盖本地, 这个在服务器有用.
git fetch --all
git reset --hard origin/master
#不要每次都输入密码, 也是在服务器用.
git config --global credential.helper cache #linux
git config --global credential.helper osxkeychain #mac
```
