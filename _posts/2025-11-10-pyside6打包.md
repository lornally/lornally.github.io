| 工具           | 平台                            | 方法             | 优点                  | 缺点                   |
| -------------- | ------------------------------- | ---------------- | --------------------- | ---------------------- |
| fbs            | Win, macOS, Linux               | 打包             | 简单 UI 安装；专注 Qt | 已弃用；更新少         |
| PyInstaller    | Win, macOS, Linux               | 打包（单文件）   | 易用，Qt 支持好       | 杀毒误报；文件大       |
| cx_Freeze      | Win, macOS, Linux               | 冻结             | 跨平台；可定制        | 配置复杂；Qt 兼容差    |
| Briefcase      | Win, macOS, Linux, iOS, Android | App 打包（Toga） | 支持移动端；标准规范  | 仅 Toga UI；学习曲线   |
| py2exe         | 仅 Windows                      | 打包             | Win 简单；无需安装    | 仅 Win；过时           |
| Nuitka         | Win, macOS, Linux               | 编译为 C++       | 运行快；混淆          | 编译慢；依赖问题       |
| pyside6-deploy | Win, macOS, Linux (Qt)          | 基于 Nuitka 打包 | 专为 PySide6；易用    | 仅 Qt；spec 文件易出错 |

pyside6-deploy 基于 Nuitka，二者打包结果接近（性能、结构），但 pyside6-deploy 更优化 Qt 依赖。
