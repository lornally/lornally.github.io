### 总结 : npm

```sh
npm install xxxx
npm update xxx
npm install moduleName ，则是将模块下载到当前命令行所在目录。
npm install moduleName -g ，模块将被下载安装到全局目录中，即Node的安装目录下的node_modules下, 但是这么这么装, 导致项目没法用. 所以, 别这么装.
```
